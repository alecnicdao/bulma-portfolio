Babysitting Website

WEB 2500

As part of WEB 2500 (User Experience Design) course I took in Fall of 2019. We were introduced to the four-step User Experience design process which includes user research, design, testing, and implementation. The following topics were covered in the course: history of user experience, user behavior, cognitive processing, personas, web analytics, content stratedy, information architecture, wrting for the web, user-centered design, usability testing, and accessibiilty. Professor Sara Lleverino then had us design a website. The website must sell a product or provide a service. In this case I chose to design a babysitting website. The reason for this was babysitting helps with peoples personal development. For instance, patience, problem-solving, communication nd relationship building, mulititasking, creative skills, and more. Also a reason is because everyone nowadays is looking for a babysitter, but people struggle on finding the right one and how to find the right one and this site does that.

WEB DEV

CS 2350 Web Project

As part of CS 2350 course I took in Spring of 2019 with Linda Duhadway we developed a webpage portfolio. Throughout the course we had to add to the webpage as we learned more about web development. We first learned about the basics of HTML. For example, how to publish onto the web. Then we learned how to do CSS (adding common style, header, navigation, etc.) Then, we did some JavaScript. For instance, how to add event listeners, how to create a form, and how to handle events on form elements. Lastly, we touched a little on jQuery. jQuery is a lightweight, "write less, do more", JavaScript library. The reason for us using it was to make it much easier to use JavaScript on your website. As for my website I talked about sports a lot, specifically basketball.
